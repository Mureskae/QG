# QG Protocol Layer

This directory defines the conceptual protocol layer of QG.

It describes how attention, presence, and resonance interact as system primitives.

---

## Core idea

QG protocol is not executable.

It defines relational rules between states of awareness and interaction.

---

## Components

### 1. Presence Layer
Defines minimal state of existence in the system.

### 2. Attention Layer
Defines directional focus and energy allocation.

### 3. Resonance Layer (PoR)
Defines mutual alignment between entities.

---

## Function of this layer

- interpret interaction
- define alignment conditions
- describe emergence patterns

---

## Note

This is a conceptual structure, not implementation.
