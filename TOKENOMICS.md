# Tokenomics uPoR-J (три токена)

### Presence Token (P)
- За факт присутствия
- Высокий объём, низкая ценность

### Resonance Token (R)
- За доказанный резонанс (включая J-space)
- Средний дефицит

### Coherence Token (C)
- За долгосрочный вклад в эволюцию J-space
- Максимальный дефицит, governance-права

**Общая эмиссия** — фиксирована.  
**Распределение** — исключительно через SNARK-доказательства uPoR-J.  
**Цель** — ценность рождается из реального внимания и резонанса, а не из спекуляции.
